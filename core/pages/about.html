каталог о компании новости контакты

Сайт "GORNIAC VAPE SHOP" является ученическим проектом. Продажа и производство жидкостей для электронных сигарет не производится.

Стек технологий:

  • Фронтенд: css и html, модальные окна - js
  • Язык: Python 3.10
  • Фреймворк: FastAPI
  • Валидация данных: Pydantic
  • Шаблоны: jinja2templates
  • База данных: PostgreSQL
  • ORM: SQLAlchemy
  • Работа с лого: Pillow
  • Проект упакован в docker-контейнер с помощью docker-compose
